# Bit Manipulation

## Introduction to Bit Manipulation

Bit manipulation deals with directly working on binary representations
(0s and 1s).\
These concepts are fundamental to understanding how computers store and
process data and are widely used in competitive programming and
system-level optimizations.

Understanding binary numbers is crucial in computer science because all
data in a computer is ultimately stored as bits.

------------------------------------------------------------------------

## Binary Number Conversion

### Decimal to Binary Conversion

A decimal number can be converted to binary by repeatedly dividing it by
2 and noting the remainders.

**Example: Convert 13 to Binary**

    13 / 2 = 6  remainder 1
    6  / 2 = 3  remainder 0
    3  / 2 = 1  remainder 1
    1  / 2 = 0  remainder 1

Reading remainders from bottom to top:

    13 (decimal) = 1101 (binary)

------------------------------------------------------------------------

### Binary to Decimal Conversion

To convert a binary number to decimal, multiply each bit by `2^position`
and sum the results.

**Example: Convert 1101 to Decimal**

    1 × 2^0 = 1
    0 × 2^1 = 0
    1 × 2^2 = 4
    1 × 2^3 = 8

**Sum = 1 + 0 + 4 + 8 = 13**

------------------------------------------------------------------------

## One's Complement and Two's Complement

### One's Complement

The one's complement of a number is obtained by flipping all bits.

**Example: One's Complement of 13**

    Binary of 13      : 0000 1101
    One's Complement : 1111 0010

------------------------------------------------------------------------

### Two's Complement

Two's complement is obtained by adding 1 to the one's complement.

**Example: Two's Complement of 13**

    One's Complement : 1111 0010
    Add 1            : 1111 0011

Two's complement is widely used to represent negative numbers in
computers.

------------------------------------------------------------------------

## Bitwise Operators

### AND (`&`)

Result bit is `1` only if both bits are `1`.

    13: 1101
     7: 0111
    & : 0101 → 5

------------------------------------------------------------------------

### OR (`|`)

Result bit is `1` if either bit is `1`.

    13: 1101
     7: 0111
    | : 1111 → 15

------------------------------------------------------------------------

### XOR (`^`)

Result bit is `1` if bits are different.

    13: 1101
     7: 0111
    ^ : 1010 → 10

------------------------------------------------------------------------

### NOT (`~`)

Flips all bits of a number.

     5 : 0000 0101
    ~5 : 1111 1010 → -6 (Two's Complement)

------------------------------------------------------------------------

## Shift Operators

### Right Shift (`>>`)

Shifts bits to the right and fills with 0s.

    13 >> 1 = 0110 → 6

------------------------------------------------------------------------

### Left Shift (`<<`)

Shifts bits to the left and fills with 0s.

    13 << 1 = 11010 → 26

------------------------------------------------------------------------

## Bit Manipulation Tricks & Techniques

### 1. Swap Two Numbers Without a Third Variable

    A = A ^ B
    B = A ^ B
    A = A ^ B

------------------------------------------------------------------------

### 2. Check if the i-th Bit is Set

    (1 << i) & num
    (num >> i) & 1

Result is non-zero if the bit is set.

------------------------------------------------------------------------

### 3. Set the i-th Bit

    num | (1 << i)

------------------------------------------------------------------------

### 4. Clear the i-th Bit

    num & ~(1 << i)

------------------------------------------------------------------------

### 5. Toggle the i-th Bit

    num ^ (1 << i)
